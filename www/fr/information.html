Les licences de NuTyX


NuTyX est regroupé sous différentes licences

Les paquets de logiciels
NuTyX est une distribution GNU/linux qui contient des logiciels écrits par un grand nombre de développeurs. Chaque paquet de logiciel est donc fourni avec sa propre licence d'utilisation choisie par son (ses) auteur(s). Pour connaître le type de licence utilisée pour un paquet particulier, veuillez consulter les codes sources dudit paquet.

Les scripts
Les scripts provenant du projet Linux From Scratch sont sous licence http://lfs.traduc.org/view/lfs...tml>Creative Commons License 2.0. Certains scripts sont ajustés pour l'usage de NuTyX par Thierry Nuttens.

Le système de maintenance
Le système de maintenance est développé depuis zéro par Thierry Nuttens et est sous licence
http://www.gnu.org/copyleft/gpl.html
