Installation

Assurez vous tout d'abord d'avoir un espace libre sur votre disque dur d'au moins 8Go ou mieux 15Go pour plus de confort.

Téléchargez la dernière version de NuTyX

La version actuelle de NuTyX s'appelle saravane.

L'iso disponible permet une installation depuis internet.Elle permet également de faire un minimum de maintenance sur la machine avant l'installation de NuTyX.

La taille des iso est +- 175 Mb.

Toutes les isos disponible peuvent être téléchargées à l'adresse :
http://downloads.nutyx.org

Créez une clé bootable

Si vous êtes déjà sous GNU/Linux :
En root
A l'aide de fdisk, déterminez où se trouve votre clé USB : 
fdisk -l
A l'aide de dd, créez votre clé bootable : 
dd if=NuTyX_x86_64-saravane.iso of=/dev/sdX valeur donnée par fdisk -l
Si vous êtes sous Windows :
Il existe surrement des logiciels pouvant le faire aisément, vous pouvez le cas échéant graver l'iso sur un CD.

Installation de NuTyX

Si vous avez choisi de compiler et installer votre propre kernel, suivez pas à pas cet article. Un minimum de connaissance sur votre matériel est recommandé pour mener à bien cette opération.

Installation de l'amorçage de la machine

Cet article vous explique comment installer depuis zero l'amorçage de la machine

Pour les plus pressé et les plus callés, voici un résumé
Si vous avez déjà grub 1 installé, il suffit juste de modifier votre menu.list selon : 
title NuTyX - Sekong sur sda2
    root (hd0,1)
    kernel /boot/kernel root=/dev/sda2 ro quiet vga=791
    initrd /boot/initrd
menuentry "NuTyX sur /dev/sda2" {
    set root=(hd0,2)
    linux /boot/kernel root=/dev/sda2 ro quiet
    initrd /boot/initrd
    }
    
    menuentry "Windows 8" {
    set root=(hd0,3)
    chainloader (hd0,3)+1
    }
