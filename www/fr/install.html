Installation

Assurez vous tout d'abord d'avoir un espace libre sur votre disque dur d'au moins 8Go ou mieux 15Go pour plus de confort.

Téléchargez la dernière version de NuTyX

La version actuelle de NuTyX s'appelle sekong.5.

L'iso disponible permet une installation depuis internet.Elle permet également de faire un minimum de maintenance sur la machine avant l'installation de NuTyX.

La taille des iso est +- 88 Mb.

La version 32 bits peut être téléchargée à l'adresse :
http://nutyx.meticul.eu/NuTyX_i686-sekong.5.iso
Il est conseillé de vérifier son intégrité avant de la graver, pour cela voici sa somme
http://nutyx.meticul.eu/NuTyX_i686-sekong.5.md5sum

La version 64 bits peut être téléchargée à l'adresse :
http://nutyx.meticul.eu/NuTyX_x86_64-sekong.5.iso
Il est conseillé de vérifier son intégrité avant de la graver, pour cela voici sa somme
http://nutyx.meticul.eu/NuTyX_x86_64-sekong.5.md5sum

Créez une clé bootable

Si vous êtes déjà sous GNU/Linux :
En root
A l'aide de fdisk, déterminez où se trouve votre clé USB : 
fdisk -l
A l'aide de dd, créez votre clé bootable : 
dd if=NuTyX_x86_64-sekong.5.iso of=/dev/sdX valeur donnée par fdisk -l

Installation de l'amorçage de la machine

Cet article vous explique comment installer depuis zero l'amorçage de la machine

Pour les plus pressé et les plus callés, voici un résumé
Si vous avez déjà grub 1 installé, il suffit juste de modifier votre menu.list selon : 
title NuTyX - Sekong sur sda2
    root (hd0,1)
    kernel /boot/kernel root=/dev/sda2 ro quiet vga=791
    
    title Maintenance de NuTyX
    root (hd0,1)
    kernel /boot/kernel ro quiet vga=791
    initrd /boot/nutyx-initrd
Si vous avez déjà grub 2 installé, il suffit juste de modifier votre grub.cfg selon : 
menuentry "NuTyX sur /dev/sda2" {
    set root=(hd0,2)
    linux /boot/kernel root=/dev/sda2 ro quiet
    }
    
    menuentry "Maintenance de NuTyX" {
    set root=(hd0,1)
    linux /boot/kernel ro quiet
    initrd /boot/nutyx-initrd
    }
    
    menuentry "Windows 8" {
    set root=(hd0,3)
    chainloader (hd0,3)+1
    }
