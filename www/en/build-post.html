    Post-construction and generation of the iso

    All the packages of a base system are now available. We can finalise and produce an installable ISO of our NuTyX

    1. We start to re-enter in the NuTyX
      2. exit
    2. We re-define if needed the LFS variable
      3. echo $LFS
      must return 
      /mnt/lfs
    3. We can now enter in the new NuTyX to finalise the job
      4. chroot $LFS /usr/bin/env -i HOME=/root TERM="$TERM" PS1='\u: \w\$' \
/bin/bash --login
      No error should be print: 
      root [ / ]#
    4. We begin by adjusting the cards.conf file. We used from now on only the recept coming from the git project. You need to adapt the language and arch if necessary (i686 for the 32 bits version).
      5. cat > /etc/cards.conf << "EOF"
dir /root/saravane/base
dir /root/saravane/system
logdir /var/log/pkgbuild
arch x86_64
locale fr
base /root/saravane/system
EOF
    5. We will save the produced binaries. To reduce the size of the iso, those will be removed.
      6. rsync -a /root/saravane/system /system

      no error should be printout

    6. We can delete the binaries from the git
      7. find /root/saravane/system/ -name "*.cards.*" -exec rm {} \;

      no error should be printout

    7. We move the logs, they don't need to be in the iso, still for a space reason
      8. mv /root/log /log
    8. We produce the squashfs files used by the iso
      9. mkdir -p /ISO/isolinux/boot/
cd /
for dir in opt bin etc lib root run sbin usr var
do
  mksquashfs $dir ISO/isolinux/boot/$dir.squashfs
done
    9. We produce an new initrd needed for the boot and the final configuration
      10. mkinitramfs
      Creating initrd... done
    10. We move the initrd and the kernel at the right place
      11. mv boot/kernel-3.18.1 ISO/isolinux/kernel
mv boot/initrd ISO/isolinux/
    11. We add the necessary files for the iso
      12. cp -av /root/saravane/iso/isolinux/* /ISO/isolinux/
      '/root/saravane/iso/isolinux/boot.cat' -> '/ISO/isolinux/boot.cat'
'/root/saravane/iso/isolinux/boot.msg' -> '/ISO/isolinux/boot.msg'
'/root/saravane/iso/isolinux/isolinux.bin' -> '/ISO/isolinux/isolinux.bin'
'/root/saravane/iso/isolinux/isolinux.cfg' -> '/ISO/isolinux/isolinux.cfg'
    12. A little simplification in the script mkiso. It will include in the next version
      13. sed -i "s/ROOTFS/LFS/" /root/saravane/scripts/mkiso
sed -i "/^LFS=/d" /root/saravane/scripts/mkiso
    13. We can now go out of NuTyX saravane and produce the iso
      14. exit
    14. We can finally generate our ISO where the version is the date of today.
      15. Before you start the next command, make shure the LFS variable is still set and the commands mkisofs and isohybrid are available on your host linux distribution. They are belonging to cdrkit and syslinux packages.
      sh $LFS/root/saravane/scripts/mkiso saravane-`date +%Y%m%d`
