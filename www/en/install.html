Typical installation of NuTyX on a single partition

Thoses screenshots will show you a typical installation on one single partition. The hard disk have a size of 8 Gbytes and used the all size for NuTyX. It means only one partition is created for the installation of NuTyX.

The size of memory is 1 Gbyte

The iso is writen on a CD. You can of course use a USB key if your machine is able to boot from it

The hard disc is totally empty, it has never been formated

  1. Bootup the usb key / CD on which you copy/burn the ISO

  2. Partition creation (optional)

    3. The partition creation is optional. If you already have an empty partition available you can directy go to the format of the partition.

  3. Format the partition (optional)

    4. The format of the partition is optional. If you do have an unformated or you want get rid of everything on it, then just format it. If you still want to keep your personnal datas for example, you then need to remove the following directories: opt, bin, etc, root, run, sbin, usr and var.

  4. Install NuTyX

    5. The installation of NuTyX doesn't need an internet.

    It takes just a few seconds

  5. Configure NuTyX

    6. Onces installed, NuTyX start immediatly without having to reboot.

    The configuration is very straight forward: network cards configuration, date and time configuration and modification of the default password of the administrator account root.

    The default password is nutyx. As soon you successfully been login first time, it will be required to change the password before you can go to the prompt.
