Make shure you have enough space on your hard disk. A size of 8Go to 15Go will make it.

Téléchargez la dernière version de NuTyX

The actual version of NuTyX is named sekong.5.

The available iso can be used to install all the packages from internel. It's also to used it for a minimum of maintenance on your machine before the actual installation of NuTyX.
The size of an iso is +- 88 Mb.

The 32 bits version can be download at:
http://nutyx.meticul.eu/NuTyX_i686-sekong.5.iso
It's highly recommanded to check it's checksum integrity before using it. Here is the md5sum of it:
http://nutyx.meticul.eu/NuTyX_i686-sekong.5.md5sum

The 64 bits version can be download at:
http://nutyx.meticul.eu/NuTyX_x86_64-sekong.5.iso
It's highly recommanded to check it's checksum integrity before using it. Here is the md5sum of it:
http://nutyx.meticul.eu/NuTyX_x86_64-sekong.5.md5sum

Create a bootable USB key

If you are currently running GNU/linux on your machine:
In root, with the help of fdisk, find out where your usb KEY is located:
fdisk -l
Then by using the command dd create a bootable key: 
dd if=NuTyX_x86_64-sekong.5.iso of=/dev/sdX valeur donnée par fdisk -l
If you are under Windows, the easiest way is to burn the ISO on a CD.

Installation of the boot process

If you used grub version 0.99, the following entry can be added to your menu.list: 
title NuTyX - Sekong sur sda2
    root (hd0,1)
    kernel /boot/kernel root=/dev/sda2 ro quiet vga=791
    
    title Maintenance de NuTyX
    root (hd0,1)
    kernel /boot/kernel ro quiet vga=791
    initrd /boot/nutyx-initrd
If you used the latest grub version 2.XX , the following entry can be added to your grub.cfg: 
menuentry "NuTyX sur /dev/sda2" {
    set root=(hd0,2)
    linux /boot/kernel root=/dev/sda2 ro quiet
    }
    
    menuentry "Maintenance de NuTyX" {
    set root=(hd0,1)
    linux /boot/kernel ro quiet
    initrd /boot/nutyx-initrd
    }
    
    menuentry "Windows 8" {
    set root=(hd0,3)
    chainloader (hd0,3)+1
    }
